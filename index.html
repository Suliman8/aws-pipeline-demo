About This Demo

This is a demonstration project showcasing AWS CodePipeline's capabilities for continuous integration and continuous deployment (CI/CD).

Pipeline Stages:

  • Stage 1 - Source: Pulls code from GitHub repository
  • Stage 2 - Build: Validates and processes code using buildspec.yml
  • Stage 3 - Deploy: Uploads files to S3 bucket for static website hosting
